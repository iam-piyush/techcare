# Patient Dashboard

## Created by Piyush Kumar

This project is a React-based patient dashboard that displays patient information such as profile, diagnosis history, lab results, and more. The dashboard features interactive charts, profile details, and icons to visually represent key health metrics.

## Features

- Patient Profile: Displaying name, date of birth, contact information, and insurance details.
- Diagnosis History: Visual representation of blood pressure (systolic and diastolic) over time.
- Lab Results: Display and download options for lab results.
- Responsive Design: The layout adjusts for different screen sizes.
- Heart Rate Monitoring: Displays heart rate information and status.
